Photographer

Masonry Layout

The project is mainly a single page application. A Masonry Layout is used to display the gallery.

The Gallery

The gallery plugin is called Lightgallery, a lightweight Javascript gallery. Lazy loading is essential, expecially in a grid of images. Basically the layout is pre-built using low-quality previews: once the full quality images are ready, they replace the preview with a nice blur animation.

Responsive

The website if fully responsive and mobile-friendly.

Restaurant

Menu

The main features of this project are the Menu page and the Reservation page.
The Menu is directly taken from a Google Sheet, which is converted to .json and then parsed. In this way, if the owner want to modify the menu, he can simply modify the Google Sheet.

Reservation

The Reservation form sends data directly to the Google Sheet, in this way the owner can easily collect all the reservations.
The Google Sheet works exactly as a CMS .
