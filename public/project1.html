Ligthweight
Gallery

The gallery plugin is called Lightgallery, a lightweight Javascript gallery. Lazy loading is essential, expecially in a grid of images. Basically the layout is pre-built using low-quality previews: once the full quality image is ready, it will replace the corresponding preview with a nice blur animation.
